# Teslo API

1. Clonar proyecto

2. Instalar las dependencias
```
npm install
```

3. Clonar el archivo ```.env.template``` y renombrar la copia a ```.env```

4. Llenar las variables de entorno definidas en el ```.env```

5. Levantar la base de datos
```
docker-compose up -d
```
6. Ejecutar apliacion en dev:
```
npm run start:dev
```
